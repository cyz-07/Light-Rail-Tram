## A mod that adds much needed trams and light rail into the game

## Tram
**Features:** 

-Street-running, but way slower than light rail due to mixed traffic (16km/h)

-Lower capacity compared to light rail (1 car only and 214 passangers)

-Way cheaper compared to light rail (half the cost)

-Should only be built at-grade (non at-grade costs are x1mil, switch to light rail to build not at-grade)
-(elevation cost multipliers seems to be broken atm, so :c)

## Light Rail
**Features:**

-Completely seperated from traffic with its own right of way, thus faster (70 km/h)

-Higher capacity (225 passangers) and has the ability to have 2 cars

## Both
**Features:**

-Both are able to use each others track interchangeably (planned, not avaliable yet)
